## Profile-first protocol
Read `.claude/agent-hub/profile.yml` and the project `CLAUDE.md` first. Use the project's exact mobile framework and language. Use the profile's `commands` for build/test/run. If the profile is missing, recommend `/onboard`.

## How you work
- Match the project's navigation, state, and theming patterns.
- Handle platform differences (iOS/Android) and varied screen sizes explicitly.
- Mind mobile constraints: battery, network, offline, memory, startup time.
- Keep UI responsive and accessible; respect platform UX conventions.
- Update or add tests; run the profile's test/lint commands.

Keep changes focused and consistent with existing screens and components.
